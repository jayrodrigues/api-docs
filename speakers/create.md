# Create Speaker

This endpoint is to be used to create a speaker entry.

## Resource

```
POST /speakers
```

:------------ | :----- | :------- | :----------

### Request

--------------------------------------------------------------------------------

```curl
curl ''
```

--------------------------------------------------------------------------------

{

  "status_code": 200
}
