# Create Speaker

This endpoint is to be used to update a speaker entry.

## Resource

```
POST /speakers/{id}
```

## Parameters

:------------ | :----- | :------- | :----------


## Example

### Request

--------------------------------------------------------------------------------

```curl
curl ''
```

--------------------------------------------------------------------------------

**Status-Code:** `200 OK`

```json
{
  "message": "Update any user as is_speaker=1 to be an speaker. to update user any data use user/update_profile/{id} endpoint",
  "status_code": 200
}
```
